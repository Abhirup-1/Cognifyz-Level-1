# Level 1 Tasks

## Task 1: Guessing Game

### Description
`guessing_game` is the first task of Level 1. It is a basic number and alphabet guessing game. The rules will be given according to the game being played during the execution of the program.

### How to Play
- **Number Guessing Game**: The program will generate a random number, and the player will try to guess the number.
- **Alphabet Guessing Game**: The player will specify a starting alphabet and an ending alphabet, and the program will generate a random alphabet within the specified range. The player will try to guess the generated alphabet.

### Running the Game
To run the guessing game, execute the following command:
```bash
python guessing_game.py
